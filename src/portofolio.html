Proyek-Proyek Saya

  • Tugas Website Penjualan Mobil

    Proyek ini adalah tugas kuliah di mana saya harus membuat sebuah website penjualan mobil. Website ini menampilkan berbagai mobil dengan spesifikasi dan harga. Pengguna dapat melihat detail mobil, menghubungi penjual, dan melakukan pemesanan.

  • Aplikasi Game Pengenalan Kendaraan

    Saya membuat sebuah aplikasi game edukasi yang mengajarkan anak-anak tentang berbagai jenis kendaraan. Dalam game ini, pengguna harus menebak nama kendaraan berdasarkan gambar yang ditampilkan.

  • Animasi Kapal

    Proyek ini adalah animasi kapal yang saya buat menggunakan software animasi. Animasi ini digunakan dalam presentasi sebuah proyek laut.

  • PBO Pembuatan Layanan Pengaduan dan Tanggapan Masyarakat

    Saya terlibat dalam pengembangan aplikasi layanan pengaduan dan tanggapan masyarakat menggunakan Pemrograman Berorientasi Objek (PBO). Aplikasi ini memungkinkan masyarakat untuk mengajukan keluhan dan mendapatkan tanggapan dari instansi terkait.
